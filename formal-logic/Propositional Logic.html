Propositional Logic

Propositional (sentential) logic (calculus) is a formal system well-formed formulas may be interpreted to represent propositions. In truth-functional propositional logic, every proposition has a truth-value of either true or false. The family of all formal systems isomorphic to truth-functional propositional logic are called zeroth-order logic.

A propositional logic is a formal system, $L$: $$L=\left(A,\Omega,Z,I\right)$$
  • $A$ (alpha set) - a finite set of propositional symbols (variables) that function as the system's atomic formulas (i.e. terminal elements) and are typically represented by the letters $P$, $Q$, $R$, etc.
  • $\Omega$ (omega set) - the finite set of operator symbols (i.e. logical connectives). $\Omega$ can be partitioned into the disjoint subsets $\Omega_0, \Omega_1, \Omega_2, ...\Omega_n$ where $\Omega_i$ represents the set of oeprators of arity j. A function's arity is the number of inputs it takes. The most common operator symbols in propositional calculus are $\neg$, $\wedge$, $\vee$, $\Rightarrow$, and $\Leftrightarrow$. Often, we treat the logical values as operators of arity zero: $\Omega_0 = \left\{0,1\right\}$.
  • $Z$ (zeta set) - a finite set of transformation rules (i.e. rules of inference)
  • $I$ (iota set) - a finite set of initial points, which are called axioms when they receive a logical interpretation
$L$ is defined inductively by
  1. Any element of $A$ is a well-formed formula of $L$.
  2. If $p_1,p_2,...p_i$ are well-formed formula and $f$ is in $\Omega_i$, then $f\left(p_1,p_2,...p_i\right)$ is a well-formed formula.
  3. Nothing else is a formula of $L$.
