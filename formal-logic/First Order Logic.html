First Order Logic

First order logic consists of a set of legal strings of symbols, partitioned into two disjoint sets: formulas and terms.

Logical Symbols

The usual logical symbols in first order logic are:
  • quantifiers: $\exists$, $\forall$
  • logical connectives: $\neg$, $\wedge$, $\vee$, $\Rightarrow$, and $\Leftrightarrow$
  • infinite set of variables: $\left\{x_0, x_1, x_2, ...\right\}$
  • parantheses: $($, $)$
  • equality: $=$
It turns out that we can give up one of the quantifiers, $\vee$, $\Rightarrow$, and $\Leftrightarrow$

Non-Logical Symbols

Non-logical symbosl are defined via the system's signature. The traditional approach is to have one, infinite set of non-logical symbols for all applications, however contemporary mathematical logic has relaxed this idea and lets the signature vary by algebra.

In first-order logic, every non-logical symbol is one of the following types:
  1. A predicate symbol with some arity≥0.
    • Propositional Variables (arity = 0) - these are true or false
    • Predicate Variables (arity ≠ 1) - these take terms and return true or false
  2. a function symbol with some arity≥0
    • constant symbols (arity = 0) - these stand for specific terms
    • function symbols (arity≥1) - these take terms and return terms

Formation Rules

The formation rules of first-order logic are usually context-free, meaning that each rule takes the form $$V \rightarrow w$$ where $V$ is a single nontermainal symbol and w is a string of symbols, which can be either terminal or nonterminal ($w$ can also be empty).

Terms

Terms are defined inductively
  1. any variable is a term
  2. any expression $f\left(t_1,...t_n\right)$ (where $f$ is a function of arity $n$ adn each argument $t_i$ is a term) is a term.

Well-Formed Formulas

The set of well-formed formulas (wff) is defined inductively as
  1. For any predicate symbol, $P$, with arity $n$ and for any set of terms $t_1,...t_n$, $P\left(t_1,...t_n\right)$ is a wff.
  2. For any terms $t_1$ and $t_2$, $t_1=t_2$ is a wff
  3. For any wff $\phi$, $\neg \phi$ is a wff.
  4. For any wffs $\phi$ and $\psi$, $\phi \Rightarrow \psi$ is a wff. Similarly for the other binary logical connectives
  5. If $\phi$ is a wff and $x$ is a variable, then $\forall x \rho$ and $\exists x \rho$ are both wff
Well-formed formulas obtained only through rules #1 and #2 are considered atomic formulas. Because of the use of parantehses, first-order logic's set of formulas have unique readability, meaning that any formula can be obtained in one and only one way via the inductive definition.

Free and Bound Variables

A variable is considered free if it is not attached to a quantifier; otherwise, it is considered bound. For instance in $$\forall x P\left(x,y\right)$$ $x$ is free and $y$ is bound.

More formally, free and bound variables are defined inductively:
  1. If $\phi$ is an atomic formula, then $x$ is free if and only if $x$ occurs in $\phi$. There are no bound variables in any atomic formula.
  2. $x$ is free in $\neg \phi$ if and only if $x$ is free in $phi$. Likewise for if $x$ is bound.
  3. $x$ is free in $\phi \rightarrow \psi$ if and only if $x$ is free in either $\phi$ or $\psi$. Likewise for bound and likewise for other binary connectives.
  4. $x$ is free in $\forall y \phi$ if and only if $x$ is free in $\phi$ and $x$ is a different symbol from $y$. Likewise for bound variales and with $\exists$.
A well-formed formula in first-order logic with no free variables is called a first-order sentence. This is equal to the set of all well-formed formulas with truth values under an interpretation.

Semantics

An interpretaiton of a first-order language
  1. determines the domain of discourse (the universe the quantifiers quantify over)
  2. assign a meaning to each non-logical constant
  3. assigns each sentence a truth value
The most common way to specify an interpretation is to specify a model (i.e. mathematical structure) consisting of a domain of discourse (a nonempty set) and an interpretation of the non-logical terms of the signature. The interpretation itself is treated as a function.
