Löb's Theorem

Statement: Consider a system with Peano arithmetic and any well-formed formula $A$. Moreover, let $P$ mean "is provable". Then: $P\left(P\left(A\right)\Rightarrow A\right)\Rightarrow P\left(A\right)$.

Explanation: In English, this reads:

If you can prove that "A being provable implies A", then A is provable.

Statement: For any first-order theory $T$ with a well-orderable language and any sentence $S$ in $T$, there is a formal proof of $S$ if and only if $S$ is a semantic consequence of $T$.

Gödel's Incompleteness Theorems

Statement: Any effective first-order theory that includes a sufficient portion of the theory of the natural numbers can never be both complete and consistent.

Explanation: In general, we care much more about consistency than completness, because an inconsistent system has sentences that are both true and false. Thus, to oversimply, it really boils down to "any interesting formal system is not complete."

Tarski's Undefinability Theorem

Statement: Let $L$ be the language of first-order arithmetic and let $N$ be its model. Every $x$ in $L$ has a Gö number $g\left(x\right)$. Let $T$ be the set of $L$-sentences true in $N$ and $T^*$ be the set of Gö numbers of these sentences. There is no formula in $L$ that defines $T^*$.

Explanation: In English, this states that no formal arithmetic cannot define the set of true statements, or, equivalently, cannot contain a function that sends all sentences to true or false correctly. However, a truth predicate can be defined in a higher order logic (e.g. second-order logic). However, this formula couldn't define its own truth predicate - that would require yet a higher logic.
