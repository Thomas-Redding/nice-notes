Modal Theory

Basics

In modal theory, a theory is a set of sentences in a formal language. An interpretation satisfies a theory if the theory is true in that interpretation. Modal theory is, essentially, the attempt to examine semantics (e.g. true and false) using syntax.

A model of a theory is a mathematical structure that satisfies the theory. For instance, consider the theory made of these axioms:
  1. 0∈N
  2. ∀x∈N x=x
  3. ∀x,y∈N (x=y)⇒(y=x)
  4. ∀x,y,z∈N (x=y)∧(y=z)⇒(x=z)
  5. ∀x∈N ∀y x=y ⇒ y∈N
  6. ∀x∈N S(n)∈N
  7. ∀x,y∈N (x=y)⇔(S(x)=S(y))
A model is a structure that follows all of these axioms. Examples include:
  • The natural numbers, where S(x)=the "successor" of x
  • The integers, where S(x)=the "successor" of x
  • The set {0, 1} where S(x)=the "successor" of x modulo 2
If we want to restrict our theory to a more specific model, we can add more axioms. For instance, this axiom
  1. ∀x S(x)≠0
eliminates models (b) and (c). Many theories have an intended model (interpretation). This, it turns out is still not enough to perfectly define the natural numbers, because its possible that, in addition to the normal natural numbers, there is another natural number, x, such that S(x)=x. In order to remove this, we can accept mathematical induction as a 9th axiom. Taken together, these axioms are known as the Peano axioms.

Universal Algebra

Universal algebra studies algebraic structures (sets with operations defined on them) as opposed to examples (models) of these structures. For instance, the axioms (1-7) designate a single algebraic structure, but with multiple models (a-c).

A formal langauge's signature is its set of non-logical symbosl together with a set of oeprations that characterize their algebraic structure. A signature can be empty, finite, infinite, infnite, or even uncountable.
