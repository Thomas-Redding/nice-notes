Probability

Basic Concepts

Events

If I have a coin and you are uncertain whether it's a trick double-heads coin or a fair coin, you could say that the world is in one of two states:

  1. The coin is double-heads
  2. The coin is fair

We call two events mutually exclusive if they can't both happen. For instance, the three events above are mutually exclusive, because only one of them can occur.

Likewise, we call a set of events exhaustive if at least one of them must happen. While we often assume a coin must land heads or tails, there are other possible outcomes such as landing on it's edge or not landing because someone caught it. Howeer, we often ignore these edge cases to simplify problems, so it's reasonable for us to say that the three events listed above are also exhaustive.

This ability to divide the world into exhaustive and mutually exclusive events is at the heart of probability theory, and the question of how to specific to make each event depends on the question you want to answer. I might split up the last into four:

  1. The coin lands heads, and it will rain tomorrow
  2. The coin lands heads, and it won't rain tomorrow
  3. The coin lands tails, and it will rain tomorrow
  4. The coin lands tails, and it won't rain tomorrow
If you just care about how the coin will land, then tomorrow's weather is irrelevant, and you might as well use the two events from before. However, if both things matter (say you've made a bet with a friend that you wouldn't wear a coat tomorrow), this new information might be useful.

Probability

If the heart of probability theory is dividing the world into mutually exclusive and exhaustive events, then the brain is realizing that we can assign these events numbers called probabilities. Although the philosophical interpretation of "probability" varies widely, everyone agrees that the numbers should correspond to how likely the event is to happen. In particular, if event $A$ has twice the probability of event $B$, then it should be twice as likely to happen.

For instance, if I roll a fair dice, then we can say there is an equal probability of each number between 1 and 6. We can represent this graphically as
1 2 3 4 5 6

Note, that just as each event has equal probability, each event has equal area. On the other hand, if you roll two dice, the table looks like
2 3 4 5 6 7
12 11 10 9 8

This makes sense, because rolling a 2 is half as likely as rolling a 3, which is half as likely as rolling a 5.

Axioms

Because drawing these tables is neither easy nor objective, it's often useful to use mathematics to make our lives easier. However, before we can use mathematics, need to things: a formal definition of probability and some notation.

To define probability, we'll have three criteria

  1. Every event must have a probability between 0 and 1.
  2. If I have a set of mutually exclusive events, then the probability of that set is equal to the sum of the individual probabilities.
  3. The probability of a exhaustive set of events is 1.

We also need some notation. We'll generally describe events in terms of random variables and values. For instance, we can represent the result of a dice roll as a random variable $D$. Unlikely normal variables, $D$ doesn't actually have a particular value that we need to solve for.

Instead, we use random variables to talk about random events. For instance, we could denote the event "I roll a 3" as $D=3$. To connect this with probabilities, I can denote "The probability I roll a 3" as $P(D=3)$.

We can treat $P(D=3)$ like a number. So, we can say that every outcome of the dice is equally likely by stating $$P(D=1)=P(D=2)=\cdots=P(D=6)$$

Similarly, we can say that a coin is biased towards heads with $$P(C=\mathrm{head}) > P(C=\mathrm{tails})$$

Knowing this notation, we can rewrite our three axioms more formally. To do this, let $\Omega$ denote a set of mutually exclusive and exhaustive events.

  1. $0 \leq P(x) \leq 1$ for all $x$ in $\Omega$
    Every event must have a probability between 0 and 1.
  2. $P(A) = \sum_{a \in A}{P(a)}$ for all $A \subseteq \Omega$
    If I have a set of mutually exclusive events, then the probability of that set is equal to the sum of the individual probabilities.
  3. $\sum_{e \in \Omega}{P(e)} = 1$
    The probability of a exhaustive set of events is 1.

More Notation

From this point forward, you should assume (unless otherwise stated) that coins and dice are fair.

In English, we can use the word "or" to combine two events. For instance, we might say "The probability that I flip a or a 5". We can represent this as $P(D=3 \cup D=5)$.
$C_1=\mathrm{tails}, C_2=\mathrm{heads}$ $C_1=\mathrm{heads}, C_2=\mathrm{heads}$
$C_1=\mathrm{tails}, C_2=\mathrm{tails}$ $C_1=\mathrm{heads}, C_2=\mathrm{tails}$

Imagine, on the other hand, that you roll two dice: $D_1$ and $D_2$. You might be interested in the probability that you roll "doubles" - that is, the same number on each dice. We can denote this $P(D_1 = D_2)$.
$C_1=\mathrm{tails}, C_2=\mathrm{heads}$ $C_1=\mathrm{heads}, C_2=\mathrm{heads}$
$C_1=\mathrm{tails}, C_2=\mathrm{tails}$ $C_1=\mathrm{heads}, C_2=\mathrm{tails}$

Finally, we probably want a way to denote that two events both happen. For instance, the probability that we rolled a "2" with the first dice and a "5" with the second dice. We can represent this simply with a comma: $P(D_1=3, D_2=5)$.
$C_1=\mathrm{tails}, C_2=\mathrm{heads}$ $C_1=\mathrm{heads}, C_2=\mathrm{heads}$
$C_1=\mathrm{tails}, C_2=\mathrm{tails}$ $C_1=\mathrm{heads}, C_2=\mathrm{tails}$

Correspondence

This notation may all seem rather etheral, so let's make it more concrete. If I have a far coin, $C$, then the probability I flip a head is 50%. That is, $P(c=\mathrm{head})=0.50$. We can represent this graphically as
$C=\mathrm{heads}$ $C=\mathrm{tails}$
Note, that the areas of both events are equal, because their probabilities are equal.
